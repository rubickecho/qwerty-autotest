## 测试功能
目前对 Qwerty Learner 的基本功能进行了测试：

* 主页面显示
* 主题切换
* 练习列表
* 词典管理
  * 语言选择
  * 分类选择
  * 词典选择
  * 章节选择
* 英语练习
  * 正确输入
  * 错误输入
  * 完成章节
  * 数据统计
